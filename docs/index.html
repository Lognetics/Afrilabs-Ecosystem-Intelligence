What's inside

Six pillars, one connected system

P

Partnerships Intelligence

Source, manage and activate every contact across Prospect → Engaged → Negotiation → Active.

M

Member Services

Onboard hubs, track payments, segment by region/tier, automate renewals across 500+ orgs.

F

Funding Flow

Grants, investments and sponsorships — Pledged → Approved → Disbursed in one ledger.

Programs OS

Accelerators, initiatives and ecosystem projects, linked to partners, funding and outcomes.

E

Events & Activation

AAG and program events from planning through execution and post-event analysis.

A

Africonnect Integration

Shared profiles, SSO, and activity sync. Three named API endpoints, bearer auth.
